# ciuy

Package for validating Uruguayan ids. Based on the ci_uy Ruby gem by Fernando Briano ([link])(https://github.com/picandocodigo/ci_uy).

Work in progress.

# Installation

```
pip install ciuy
```

# Testing

You can run tests from the project root after cloning with `nose2`.
